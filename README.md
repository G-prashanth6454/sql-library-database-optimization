# Library Database Optimization (SQL)

## Project Overview
This project focuses on designing and optimizing a relational database for a library management system. The goal was to ensure data integrity, improve query performance, and support real-world reporting requirements.

## Tools & Technologies
- MySQL
- SQL
- MySQL Workbench

## Key Features
- Designed a normalized database schema (3NF)
- Implemented primary and foreign keys with cascading constraints
- Created indexes to improve query performance
- Enforced referential integrity across multiple tables

## Database Tables
- Books
- Authors
- Borrowers
- Book Copies
- Loans
- Library Branches

## SQL Concepts Used
- Complex JOINs
- GROUP BY, HAVING
- Subqueries
- Window Functions
- Views
- Indexing

## Sample Use Cases
- Tracking branch-wise book loans
- Identifying borrowers with more than 5 books issued
- Counting available copies across branches
